# Privacy Policy

Last updated: March 15, 2026

## Overview

FlowTab is a personalized new tab extension for Chrome and Edge.
FlowTab is designed with a local-first approach and stores user settings in the local browser profile.

## Data Storage

FlowTab stores configuration data locally in the browser profile.
This may include:

- widget configuration
- quick links
- notes and messages entered by the user
- background preferences
- layout and appearance settings
- import and export configuration data

FlowTab does not require a user account to function.

## Data Transmission

FlowTab does not send user settings to developer-operated servers.

Some optional features may send user-initiated requests to third-party services in order to function, for example:

- search requests sent to the selected search provider
- background image requests sent to Unsplash if that background source is enabled
- weather or other widget-related requests sent to the relevant third-party service if the widget is enabled

These requests are only used to provide the requested feature.

## Import and Export

FlowTab allows users to export and import their settings manually.
Exported settings are created locally by the extension and remain under the user's control unless the user chooses to share or back them up elsewhere.

## Error Reporting

Public FlowTab builds disable Sentry error reporting by default.

If a custom build enables error reporting by explicitly configuring a `SENTRY_DSN`, error information may be sent to the configured Sentry project for debugging purposes.
This is optional and is not enabled by default in the standard build.

## Remote Code

FlowTab does not download or execute remotely hosted code.
All executable extension logic is packaged with the extension itself.

## Data Sales

FlowTab does not sell user data.
